4. ความยากลำบากในการผลิต (Physical Proof-of-Work: PPoW)

การ์ดทุกใบถูกสร้างขึ้นด้วยมือ (Handmade) และมีเนื้อหาอ้างอิงตามเหตุการณ์จริงในช่วงเวลานั้น ข้อจำกัดด้านแรงงานและเวลา (PPoW) นี้ ช่วยป้องกันการปั๊มการ์ดจำนวนมาก (Anti-spam) และทำให้การ์ดแต่ละใบมีเอกลักษณ์ที่ไม่สามารถผลิตซ้ำได้

5. เครือข่ายการกระจายข้อมูล (Network Distribution)

ข้อมูลของการ์ดทุกใบจะถูกบันทึกและเผยแพร่ลงบนโปรโตคอล Nostr โดยตรงเพื่อทำหน้าที่เป็น Public Ledger ที่มีความกระจายศูนย์ (Decentralization) ปลอดภัยจากการถูกลบหรือแก้ไข โดยมีเกม PMATCGV ในการดึงข้อมูลและเผยแพร่ไฟล์การ์ด นอกจากนี้ ระบบยังเปิดให้ดาวน์โหลดไฟล์ในรูปแบบ Print-and-Play เพื่อให้ผู้เล่นนำไปพิมพ์เล่นเองได้ฟรี กลไกนี้สร้างประโยชน์ร่วมกันแบบ Win-Win โดยผู้เล่นสามารถเข้าถึงความสนุกได้ทันที ส่วนผู้สร้างก็ได้กระจายผลงานสู่สังคม

อุปกรณ์: การ์ด Node ฝั่งละ 1 ใบ, ลูกเต๋า 6 หน้า (D6) ฝั่งละ 3 ลูก

//SET UP

Sync Phase (เริ่มเกม): นำค่า SPD ของ Node ทั้งสองฝ่ายมาเปรียบเทียบกัน ฝ่ายที่ SPD สูงกว่าจะได้เป็น ฝ่ายรุก (Attacker) ก่อน (หากเท่ากัน ให้ทอยเต๋าเพื่อตัดสิน)

//START GAME >>> TURNการเล่นจะแบ่งเป็น 4 ช่วง (Phases) ดังนี้:

1. Attack Phase (ฝ่ายรุก):

  • Dice Roll (ทอยเต๋า): ฝ่ายรุกทอยลูกเต๋า 3 ลูก (กฎตอง: หากออกหน้าเดียวกัน 3 ลูก +3Spd (1Turn)) ผู้เล่นสามารถจ่าย 2 HP เพื่อได้รับ 2 PoW เพิ่มได้ (1ครั้ง/turn) (Overclock)
  • Activate (เปิดสถานะ Node): เลือกสละลูกเต๋ากี่ลูกก็ได้ เพื่อนำผลรวมหน้าเต๋ามาสร้างเป็นแต้ม PoW นำแต้ม PoW ไปจ่ายค่า BPoW เพื่อเปลี่ยน Node ให้อยู่ในสถานะ Online (พร้อมสั่งการ)

    • Command: ออกคำสั่งให้ Node โจมตีเป้าหมาย หรือ Offline

    • Dodge: สละลูกเต๋า(ที่ยังไม่ถูกใช้สร้างPoW) 1ลูก เพื่อให้ Node สามารถเลือกตั้งรับด้วย Dodge ได้
    • Command: ออกคำสั่งให้ Node ตั้งรับ(เลือกได้ 1 อย่าง) หรือ Offline:
      • Defend (ป้องกัน): ใช้ค่า DEF รับการโจมตี
      • Dodge (หลบหลีก): ทำได้เมื่อผ่านเงื่อนไข Boost Dodge มา

3. Settle Phase (สรุปผล):

คำนวณความเสียหายตามที่ฝ่ายรับเลือก (ถ้า att ฝ่ายรุก ≤ 0 ให้ถือว่า damage = 0):

  • หากเลือก Defend (และ Offline): ความเสียหาย (Damage) = ATT ของฝ่ายรุก - DEF ของฝ่ายรับ (หาก ATT หักลบ DEF แล้ว ≤ 0 จะรับดาเมจขั้นต่ำ 1 Damage)

  • หากเลือก Dodge: ผู้เล่นทั้งสองฝั่งทอยเต๋าคนละ 1 ลูก นำแต้มที่ได้บวกกับค่า SPD ของตนเอง
    • รุก > รับ (หลบพลาด): รับความเสียหายเต็มจำนวน (Damage = ATT ฝ่ายรุก โดยไม่นำ DEF มาหักลบ)
    • รุก < รับ (หลบสำเร็จ): ไม่ได้รับความเสียหาย (Damage = 0)

4. End Phase:

  • ล้างสถานะ (Clear): ยกเลิกการปรับเปลี่ยน Stat ชั่วคราวทั้งหมดในเทิร์นนี้ (คงไว้เฉพาะผลถาวร) สำหรับสถานะผิดปกติ (Status Ailments) ให้คงอยู่หรือถูกล้างออกตามเงื่อนไขเฉพาะของสถานะนั้นๆ
  • สลับบทบาท: ฝ่ายรับสลับมาเป็นฝ่ายรุก และกลับไปเริ่ม Attack Phase ใหม่อีกครั้ง
  • Shutdown: หาก HP ของ Node ฝ่ายใดลดเหลือ 0 จะถือว่าถูก Shutdown และเอา Node นั้นออกจากสนามทันที ผู้เล่นใดไม่มี Node ให้เล่นให้ถือว่าเป็นผู้พ่ายแพ้ทันที ส่วนผู้ชนะจะได้รับ 1 Point และเริ่มเล่นเกมกันใหม่อีกรอบ
  • Game Over: ผู้เล่นที่มี 2 Point ก่อน จะถือว่าเป็นผู้ชนะในเกมนี้ไป

Condition (สถานะ):

  1. Online: The controller can Boost & Command this Node.
  2. Offline: The controller cannot Boost & Command this Node. Its DEF is halved (rounded up).

Status Ailments:

  • Muted: ไม่สามารถใช้สกิล (SPoW) ได้ (หายไปเมื่อเข้าสู่สถานะ Offline)
  • Overheat: ไม่สามารถใช้ Overclock ได้ (หายไปเมื่อเข้าสู่สถานะ Offline)
  • 51%Att: เมื่อการ์ดติดสถานะ Offline จะโดนดาเมจ 1 HP ทันที (หายไปเมื่อเข้าสู่สถานะ Online)
  • Bug: สลับค่า ATT(พื้นฐาน) และ DEF(พื้นฐาน) ของตนเอง (หายไปเมื่อจบเทิร์น)
  • Virus: สุ่มรับ 1 สถานะผิดปกติ (หรือไม่มีผล)
  • Asleep: บังคับเป็นสถานะ Offline และยกเลิกการโจมตี การ์ดจะตื่น (หายจากสถานะนี้) เมื่อได้รับดาเมจ หรือ ตื่นเองเมื่อจบเทิร์น

10. สูตรคำนวณ (Formula)

การ์ด Node ทุกใบถูกกำหนดค่า HP, Att, Def และ Spd มาจากข้อมูล Bitcoin Blockchain

  • HP มาจากเลขหลักหน่วยของ Block Height(X)
  • Att มาจากเลขหลักสิบของ Block Height(X)
  • Def มาจากเลขหลักร้อยของ Block Height(X)
  • Spd มาจากการหาเลข 1-3 ในค่า Hash ของ Block Height นั้น โดยเริ่มจากตำแหน่งสุดท้ายก่อนแล้วค่อยขยับขึ้นมา ถ้าหาเลขไหนเจอก่อนก็จะใช้เลขนั้น(1-3) เป็นค่า spd (ถ้าหาไม่ได้ spdจะ = 0)

หลังจากนั้นจะเข้าสู่สูตรคำนวณเพื่อหา ค่าร่าย (PoW) สายอาชีพ (Class) และ ธาตุ (Element)

10.1 สูตรคำนวณค่าร่าย (BPoW Calculation Formula)

สูตรออกแบบมาเพื่อรักษาสมดุล (Balance) การ์ดที่สเตตัสสูงจะต้องใช้แต้มลูกเต๋าในการเล่นสูงตาม ในขณะที่การ์ดสเตตัสต่ำจะสามารถสั่งการหรือใช้สกิลได้ง่ายกว่า

8. Gameplay Protocol

8.1 1V1 Game Mode (1PMA = 1PMA)

Objective: Reduce the opposing Node's HP to 0.

Equipment: 1 Node card per side, 3 six-sided dice (D6) per side.

//SET UP

Sync Phase (Game Start): Compare the SPD stats of both Nodes. The side with higher SPD becomes the Attacker first. (If tied, roll dice to decide).

//START GAME >>> The turn is divided into 4 Phases:

1. Attack Phase (Attacker):

  • Dice Roll: Attacker rolls 3 dice (Triple Rule: If all 3 dice show the same number, +3 SPD for 1 Turn). Players can pay 2 HP to gain 2 PoW (Overclock, max 1 time/turn).
  • Activate (Turn Node Online): Sacrifice any number of dice to sum their values as PoW. Pay PoW equal to BPoW to change the Node to Online status (ready for command).
    Note: If PoW is insufficient, the Node is Offline: Cannot Boost & Command, and DEF is halved (rounded up).
  • Boost (Optional):
    • Active Skill (SPoW): Use remaining PoW to pay the SPoW cost and activate the skill (Limit 1 time/skill).
    • Dash: Sacrifice 1 unused die to give the Node Dash status (Total Attack = ATT + SPD).
    • Command: Order the Node to Attack the target or go Offline.

It is a "never-ending battle" as long as we believe there is a mirror (an Ego) to maintain, and dust (defeat/pain) to sweep away.

11.2 The Sloka of Emptiness (The State of Awakening)

When Huineng (Eno), the 6th Patriarch, heard the first Sloka, he composed a counter-Sloka. This Sloka "smashes the board" where the three elements are fighting into pieces:

"Fundamentally no Bodhi-tree exists, nor the frame of a mirror bright. Since all is void from the beginning, where can the dust alight?"

Interpretation in PMATCG (Huineng's Path): In truth, there is no mirror (no Awareness that needs to be forced) and no dust (no Ego or suffering with substance). When there is no "self" for the dust to alight upon, the cycle of the biting struggle between Awareness, Suffering, and Ego is broken.

This aligns with the ultimate teaching of the Zen master's great-grand-disciple, pointing to the highest essence:

"All the Buddhas and all sentient beings are nothing but the One Mind, beside which nothing exists... Using the mind to seek Mind is the greatest mistake. Only when conceptual thinking is halted will the Original Mind appear." - Master Huangbo
